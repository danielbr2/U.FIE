\hypertarget{working-with-jupyter-notebook}{%
\section{Working with Jupyter
Notebook}\label{working-with-jupyter-notebook}}

The following example is based on Script \texttt{Descr-Figures} from
Chapter 1 and demonstrates the use of \textbf{Jupyter Notebooks} to
document your work step by step. We will describe the two most important
building blocks:

\begin{itemize}
\tightlist
\item
  basic Markdown commands to format your text in \texttt{Markdown} cells
\item
  how to import and run Julia code in \texttt{Code} cells
\end{itemize}

\hypertarget{import-and-prepare-data}{%
\subsection{Import and Prepare Data}\label{import-and-prepare-data}}

Let's start by loading all packages:

\begin{Shaded}
\begin{Highlighting}[]
\ImportTok{using} \BuiltInTok{WooldridgeDatasets}\NormalTok{, }\BuiltInTok{Statistics}\NormalTok{, }\BuiltInTok{DataFrames}\NormalTok{, }\BuiltInTok{FreqTables}\NormalTok{, }\BuiltInTok{Plots}
\end{Highlighting}
\end{Shaded}

In the next step, we import our data and define important variables:

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{affairs }\OperatorTok{=} \FunctionTok{DataFrame}\NormalTok{(}\FunctionTok{wooldridge}\NormalTok{(}\StringTok{"affairs"}\NormalTok{))}
\NormalTok{counts }\OperatorTok{=} \FunctionTok{freqtable}\NormalTok{(affairs.kids)}
\NormalTok{labels }\OperatorTok{=}\NormalTok{ [}\StringTok{"no"}\NormalTok{, }\StringTok{"yes"}\NormalTok{]}

\FunctionTok{print}\NormalTok{(counts)}
\end{Highlighting}
\end{Shaded}

\begin{verbatim}
[171, 430]
\end{verbatim}

\hypertarget{calculate-descriptive-statistics}{%
\subsubsection{Calculate Descriptive
Statistics}\label{calculate-descriptive-statistics}}

Now we are interested in printing out the average age. We start with its
definition and use LaTeX to enter the equation:
\[ \bar{x} = \frac{1}{N} \sum_{i=1}^N x_{i} \] The resulting Julia code
gives:

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{age\_mean }\OperatorTok{=} \FunctionTok{mean}\NormalTok{(affairs.age)}
\FunctionTok{print}\NormalTok{(age\_mean)}
\end{Highlighting}
\end{Shaded}

\begin{verbatim}
32.48752079866888
\end{verbatim}
